### Running Containers on Amazon EKS 과정의 목표

* API의 핵심 객체 및 주요 구성 요소를 포함하여 Kubernetes의 기본 구성 요소에 관해 설명합니다.
* Amazon EKS가 Kubernetes 제어 영역과 데이터 영역의 일부를 관리하는 방법을 설명합니다.
* Amazon EKS 클러스터를 구축하고 유지 관리합니다.
* Amazon EKS 클러스터에 애플리케이션을 배포합니다.
* 엔터프라이즈 규모로 Amazon EKS 클러스터에서 실행되는 애플리케이션을 관리합니다.
* 효율적이고 안전한 통신을 구성합니다.
* Amazon EKS 클러스터에서 관찰 기능을 구성합니다.
* Amazon EKS에서 실행되는 애플리케이션을 위한 스토리지를 프로비저닝합니다.
* Amazon EKS 클러스터를 보호합니다.
